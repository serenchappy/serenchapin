StrongBeenz E-Commerce Site

For my Software Developement class at CU Boulder, I was challenged to create a software project of some sort. My love of web-design pushed into creating an e-commerce website. This site allows you to create an account/login, add items to your cart, and to purchase those items.

Figma

Using Figma I created a wireframe to get a better idea of how I wanted to look. Everything from selecting the correct font to picking complementary colors that enhanced the site and its products.

Photoshop

Though this site doesn't have actual products to sell, I wanted the pictures of the hypothetical products to look real themselves to make the website look more professional.

Backend

PythonFlask Flask is a small and lightweight Python web framework that provides useful tools and features that make creating web applications in Python easier. It gives developers flexibility and is a more accessible framework for new developers since you can build a web application quickly using only a single Python file.
